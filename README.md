# AI Multi-Agent Research Assistant

This project is an AI-powered multi-agent system that can:
- Search research papers (Arxiv)
- Get information from Wikipedia
- Perform web search
- Answer questions using LLMs

## Tech Stack
- Python
- FastAPI
- Streamlit
- LangChain

## How to Run
1. Create virtual environment
2. Install requirements
3. Run backend using FastAPI
4. Run UI using Streamlit
